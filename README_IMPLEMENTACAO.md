# Passo a passo de publicação e commit

1. Extraia este pacote.
2. Envie todo o conteúdo para a raiz do repositório GitHub `geracard`.
3. No Netlify, conecte o repositório.
4. Configure as variáveis do `.env.example` no painel do Netlify.
5. Faça o primeiro deploy.
6. Teste as URLs:
   - `/api/health`
   - `/api/load-records`
7. Edite um registro no painel e salve.
8. Verifique se um novo commit foi criado automaticamente no GitHub.
